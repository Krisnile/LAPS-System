# LAPS-System
Design of a Lightweight Annotation System Supporting Prompt-Driven Segmentation

This system is designed to create a simple and efficient image annotation platform that integrates existing deep learning models such as the **Segment Anything Model (SAM)**. It supports automatic segmentation of image regions through **point or box prompts**, reducing the burden of manual annotation and achieving a **semi-automated annotation workflow**. The system supports basic functions such as image uploading, interactive segmentation, result visualization, and export of annotation data. The backend uses the **Django framework** to implement user management and data storage interfaces, facilitating front-end and back-end collaborative development. This system is intended for everyday research and experimental data preparation scenarios, focusing on **practical functionality, a clean interface, and ease of use of the model**, making it suitable as an auxiliary tool for image segmentation data annotation tasks.
